The Pytorch code of the paper "[Selectivity Matters: Source Node Influence Pruning
for Unsupervised Graph Domain Adaptation](./manuscript-SNIP.pdf)".


## requirements

```
pip install torch-scatter==2.0.9 torch-sparse==0.6.13 torch-cluster==1.6.0 torch-spline-conv==1.2.1 -f https://data.pyg.org/whl/torch-1.11.0+cu113.html
pip install torch-geometric==2.1.0
pip install cvxopt
pip install networkx
pip install python-louvain
pip install seaborn
```
